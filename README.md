# Pandas-python-data-visualization
This project demonstrates how to create common data visualizations using Python and Pandas. It includes line charts, bar charts, pie charts, histograms, scatter plots, box plots, and stacked charts using a single Python script.
# Pandas Data Visualization in Python

This project showcases how to generate multiple types of charts using Python Pandas and Matplotlib.

## Charts Covered
- Line Chart
- Bar Chart
- Multi-line Chart
- Pie Chart
- Histogram
- Scatter Plot
- Box Plot
- Stacked Bar Chart

## Technologies Used
- Python
- Pandas
- Matplotlib
- Colab

## Use Cases
- Data analysis practice
- Learning Pandas visualization
- Interview preparation
- Beginner-friendly reference

## How to Run
1. Install dependencies:
   pip install pandas matplotlib
2. Run the script:
   python pandas_charts.py
